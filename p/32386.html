Windows总结一【CMD常用命令详解】

发表于2025-06-07|更新于2025-06-21|Window
|浏览量:|评论数:

文件与磁盘操作

命令 说明
md 新建文件夹 新建目录
mkdir 新建文件夹 同上
dir 列出当前目录内容
dir /s /ah 递归显示目录及隐藏文件
cd 目录名 进入指定目录
cd .. 返回上一级
cd /d D:\logs 跨盘符切换目录
robocopy C:\src D:\backup /MIR /MT:8 多线程镜像备份
del 文件名 删除文件
del /f /q /s 文件名 强制/静默/递归删除
*del /F /Q .tmp 强制删除临时文件
rd /S /Q “D:\old” 无提示删除非空目录
fsutil file createnew test.txt 1048576 生成1MB测试文件
type filename.txt 显示文本文件内容
copy file1.txt+file2.txt merged.txt 合并文件
xcopy C:\data D:\backup /E /H /C 复制目录树
move file.txt D:\newfolder 移动文件
ren old.txt new.txt 重命名文件
attrib +h secret.txt 设置隐藏属性
fc file1.txt file2.txt 文件内容比较
find “error” log.txt 文件中搜索字符串
tree /F 显示目录树结构
md newfolder 创建新目录
cipher /W:D: 彻底擦除磁盘剩余空间
compact /c /s 启用NTFS压缩
diskpart 启动磁盘分区工具
format E: /FS:NTFS /Q 快速格式化磁盘

系统管理与信息查看

命令 说明
systeminfo 系统信息
hostname 查看机器名
ver 操作系统版本
tasklist 当前进程列表
taskkill /PID 进程号 /F 结束进程
wmic cpu get Name CPU信息
wmic memorychip get capacity 查看计划任务
schtasks /query /fo LIST 内存信息
set 环境变量
cleanmgr 启动磁盘清理工具
msinfo32 打开系统信息面板
winver 显示Windows版本
services.msc 服务管理控制台
compmgmt.msc 计算机综合管理
devmgmt.msc 设备管理器
eventvwr 事件查看器
perfmon 性能监视器
taskschd.msc 任务计划程序
lusrmgr.msc 本地用户和组管理
control 打开控制面板
appwiz.cpl 程序和功能面板
sysdm.cpl 系统属性设置
secpol.msc 本地安全策略
rsop.msc 组策略结果集
slmgr.vbs -xpr 检查系统激活状态
wmic qfe list 查看已安装补丁
wmic bios get serialnumber 获取BIOS序列号
powercfg /energy 生成电源效率报告
time 显示或设置系统时间
date 显示或设置系统日期
shutdown /r /t 0 立即重启系统
shutdown /s /t 0 立即关机

磁盘和文件系统工具

命令 说明
diskmgmt.msc 磁盘分区管理
chkdsk C: 检查磁盘
chkdsk /f 检查磁盘错误并修复（需重启） [管理员]
diskpart 硬盘分区工具
format D: 格式化磁盘
vol 显示卷标与序列号
fsutil 文件系统高级操作
sfc /scannow 扫描并修复系统文件损坏 [管理员]
defrag C: /O 文件系统高级操作
fsutil 优化机械硬盘碎片（SSD无需使用）

网络操作与排查

命令 说明
ipconfig /all 显示完整网络配置
ipconfig /release 释放当前IP
ipconfig /renew 重新请求IP
ipconfig /flushdns 清除DNS缓存
ipconfig /registerdns 刷新DNS注册
ping www.baidu.com 测和百度的网络是否通畅
ping 192.168.1.1 路由器/网关连通
ping 域名/IP -n 10 连续发10个包
ping 8.8.8.8 -t 持续测试网络连通性
tracert www.baidu.com 路由追踪
tracert -d www.baidu.com 跳过DNS解析
pathping www.baidu.com 跳点延时及丢包率分析
netstat -an 所有监听端口、连接
netstat -ano 再带上进程PID
netstat -ab 一个进程/应用占哪些端口
netstat -ano *| findstr ESTABLISHED 揪出电脑与陌生IP的隐蔽连接
nslookup 进入交互模式，能查A记录/MX等
nslookup www.qq.com DNS解析验证
arp -a 显示ARP缓存表
netsh interface ip show config 显示网卡配置
netsh interface ip set dns “以太网” static 8.8.8.8 设置静态DNS
netsh winsock reset 重置网络协议栈
netsh advfirewall set allprofiles state off 临时关闭防火墙
netsh wlan show profiles 查看保存的WiFi配置
netsh wlan show profile name=”Home” key=clear 查看WiFi密码
netsh wlan connect ssid=”Office” 连接指定WiFi
netsh trace start capture=yes 启动网络抓包
netsh trace stop 停止网络抓包
route print 显示路由表
route add … 添加路由
route delete … 删除路由
route print 查看路由表
net use 查看网络映射驱动器
net use K: \192.168.1.100\share 映射网络驱动器
pathping www.taobao.com 综合路由和丢包测试
getmac /v 显示MAC地址详情
net view 查看局域网共享资源
net share 管理共享文件夹
ftp 启动FTP客户端
telnet 启动Telnet客户端
scp root@192.168.8.132:/root/test002 ./Desktop scp root@服务器ip:文件绝对路径
scp Desktop\test003.txt root@192.168.8.132:/root/ scp Desk\文件路径 root@服务器ip:文件绝对路径

进程与用户管理

命令 说明
tasklist /svc 查看进程及关联服务
taskkill /F /IM chrome.exe 强制结束进程
taskkill /PID 1234 /T 结束进程树
net start 服务名 启动服务
net stop 服务名 停止服务
start notepad 启动应用程序
net user Tech P@ssw0rd /add 创建用户
net localgroup administrators Tech /add 提权为管理员
net user Tech /delete 删除用户
query session 查看远程桌面会话
whoami 显示当前用户
runas /user:admin cmd 以其他用户身份运行
qwinsta 显示终端服务会话（同query session）
sc query 查看服务状态
sc stop WinDefend 停止服务
sc config DiagTrack start= disabled 禁用服务
wmic process get name,processid 获取进程列表

运维实战命令

命令 说明
for /L %i in (1,1,100) do ping -n 1 192.168.1.%i 批量Ping扫描
**forfiles /p “C:\logs” /s /m .log /d -7 /c “cmd /c del @path”* 自动清理日志
auditpol /set /category:”Account Logon” /success:enable 启用登录审核
wmic product get name,version > software.csv 导出软件清单
wmic memorychip get capacity,speed 获取内存信息
wmic diskdrive get model,size 获取硬盘信息
reg query “HKLM\Software\Microsoft\Windows\CurrentVersion\Run” 查看启动项
schtasks /query /fo LIST /v 查看计划任务详情
vssadmin list shadows 查看卷影副本
powercfg /batteryreport 生成电池健康报告

高危操作预警

命令 说明
format C: /FS:NTFS :: 永久删除命令（不可恢复）
diskpart → clean :: 永久删除命令（不可恢复）
del /F /S /Q . :: 谨慎使用的命令
rd /S /Q C:\Windows :: 谨慎使用的命令
