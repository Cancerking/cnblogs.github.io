Windows总结三【超实用快捷键】

发表于2025-06-18|更新于2025-06-21|Window
|浏览量:|评论数:

基础操作

命令 说明
Ctrl + C 复制选中内容
Ctrl + X 剪切选中内容
Ctrl + V 粘贴
Ctrl + Z 撤销上一步操作
Ctrl + Y 恢复撤销的操作
Ctrl + A 全选当前内容
Ctrl + H 替换 (在文档中查找并替换)
Delete 删除选中项（到回收站）
Shift + Delete 永久删除（不经过回收站）
Ctrl + S 保存当前文档或文件
Ctrl + P 打印当前文档
Ctrl + O 打开文件
Ctrl + N 新建文件或窗口
Home / End 将光标移动到行首 / 行尾
Ctrl + Home / Ctrl + End 将光标移动到文档开头 / 文档末尾
Ctrl + ← / Ctrl + → 按单词向左/向右移动光标
Ctrl + Backspace / Ctrl + Delete 删除光标前/后的一个单词
Shift + 方向键 选中文本 (可结合 Ctrl 选中单词或段落)

窗口管理

命令 说明
Alt + Tab 切换打开的程序窗口
Win + Tab 打开任务视图（虚拟桌面管理）
Win + ←/→ 将窗口贴靠到屏幕左/右侧（分屏）
Win + ↑/↓ 最大化/最小化当前窗口
Win + D 显示桌面（最小化所有窗口）
Win + M 最小化所有窗口
Win + Shift + M 还原最小化的窗口
Alt + F4 关闭当前窗口或程序
Win + Ctrl + D 创建新的虚拟桌面
Win + Ctrl + ← / Win + Ctrl + → 在虚拟桌面之间切换
Win + Ctrl + F4 关闭当前虚拟桌面

文件资源管理器

命令 说明
Win + E 打开文件资源管理器
Ctrl + Shift + N 新建文件夹
F2 重命名选中文件/文件夹
F3 在资源管理器中搜索
Alt + Enter 显示选中文件的属性

系统快捷操作

命令 说明
Win + L 锁定电脑
Win + E 打开「文件资源管理器」
Win + I 打开「设置」应用
Win + R 打开“运行”对话框
Win + I 打开系统设置
Win + A 打开「通知中心」/「操作中心」
Win + X 打开「高级用户菜单」
Win + S 打开搜索栏（全局搜索）
Win + P 切换投影模式（多显示器）
Win + Tab 打开「任务视图」 (查看所有窗口和虚拟桌面)
Win + Pause/Break 快速打开「系统属性」
Ctrl + Shift + Esc 直接打开任务管理器
Win + X 打开高级用户菜单（快速访问工具）

浏览器 & 文本编辑

命令 说明
Ctrl + T 新建浏览器标签页
Ctrl + N 打开新的资源管理器窗口
Ctrl + W 关闭当前资源管理器窗口或标签页
Ctrl + E 将光标定位到地址栏或搜索框
Ctrl + Shift + T 恢复刚关闭的标签页
Ctrl + F 页面内查找
Alt + D 定位到地址栏并选中路径
F5 / Ctrl + R 刷新页面
Ctrl + Home/End 跳转到文档开头/结尾
Ctrl + Shift + ←/→ 按单词选中文本
Ctrl + F5 / Ctrl + Shift + R 强制刷新
Ctrl + D 将当前网页添加到收藏夹/书签
Ctrl + H 打开历史记录
Ctrl + J 打开下载列表

其他实用快捷键

命令 说明
Print Screen 截取全屏（保存到剪贴板）
Win + Print Screen 截屏并自动保存到“图片”文件夹
Win + Shift + S 区域截图（支持矩形/自由形状）
Win + V 打开剪贴板历史记录（需启用）
Win + . 打开表情符号面板
